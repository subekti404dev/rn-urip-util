## REACT NATIVE KIT BY URIP

### Installation
```bash
npm install github:subekti404dev/urip-rn-kit

or

yarn add github:subekti404dev/urip-rn-kit
```

### Example Usage
```jsx
import {Row, Col} from 'urip-rn-kit'

render() {
  return (
    <Row>
      <Col></Col>
      <Col size={2}></Col>
    </Row>
  )
}

```
