## REACT NATIVE URIP UTIL

### Installation
```bash
npm install github:subekti404dev/rn-urip-util

or

yarn add github:subekti404dev/rn-urip-util
```

### Example Usage
```jsx
import {Row, Col} from 'rn-urip-util'

render() {
  return (
    <Row>
      <Col></Col>
      <Col size={2}></Col>
    </Row>
  )
}

```
